### 1.'A'和"A"有什么区别？  
'A'表示的是字符常量char，"A"表示的是字符串string。  

### 2.简述C语言程序的编译过程。
1. 预处理：删除所有注释，生成后缀为(.i)的临时文件。  
2. 编译：用内置编译器软件将(.i)临时文件转换为具有汇编级指令(低级代码)的汇编文件。  
3. 汇编：使用汇编程序将程序集级代码(.s文件)转换为机器可理解的代码(二进制/十六进制形式)。  
4. 连接：将库文件包含在我们的程序中。  

### 3.为防止头文件被多次包含，应该怎么做？  
使用头文件保护代码。  
在头文件开头加上：
```c
#ifndef name
#define name
```
然后再末尾加上:
```c
#endif
```
